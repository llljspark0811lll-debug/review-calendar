# Review Calendar Web

## Local run

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3005
```

## Scripts

- `npm run dev` : start development server on port `3005`
- `npm run build` : create production build
- `npm run start` : start production server on port `3005`
- `npm run lint` : run ESLint
