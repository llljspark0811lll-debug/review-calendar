# 시장조사 메모 - 경쟁 서비스 초안

기준일: 2026-06-02

## 1. 결론 요약

공개적으로 확인되는 서비스는 크게 두 부류였다.

- 체험단 공고를 여러 사이트에서 모아 보여주는 통합 앱/사이트
- 크리에이터의 브랜드 협업, 일정, 마감일, 딜 파이프라인을 관리하는 해외 SaaS

반면 현재 구상 중인 서비스처럼 아래를 동시에 제공하는 사례는 쉽게 확인되지 않았다.

- 국내 체험단 사이트 여러 곳 대응
- 사이트별 계정 연동
- 로그인된 상태에서 선정 상세 링크 파싱
- 체험 가능 기간과 리뷰 마감일 자동 수집
- 캘린더에서 일정 배정

즉, 부분적으로 겹치는 서비스는 있지만 `정확히 같은 형태의 직접 경쟁 서비스`는 공개 검색 기준으로는 뚜렷하지 않다.

## 2. 국내에서 보인 유사 서비스

### 체리단

구글 플레이에 등록된 `체리단`은 여러 플랫폼의 체험단 공고를 모아 보여주고, 찜, 신청 이력, 선정 여부, 리뷰 작성 여부를 관리하는 앱으로 소개된다.

확인된 특징:

- 다양한 플랫폼의 체험단 공고 통합
- 찜 기능
- 신청/선정/리뷰 현황 관리
- 방문/리뷰 마감일 표시

한계로 보이는 점:

- 실제 신청은 각 체험단 사이트에서 이루어진다고 명시
- 공개 설명만 보면 `선정 링크를 넣어 로그인 상태에서 상세 정보를 자동 파싱`하는 구조는 확인되지 않음

출처:

- Google Play `체리단 – 체험단 공고 통합 모음앱`

## 3. 국내에서 보인 인접 서비스

### 협찬플러스

`협찬플러스`는 체험단 공고를 한곳에 모아주고, 일정, 가이드, 메모 관리까지 돕는 형태로 보인다. 다만 공개 페이지 설명상 제품 성격은 `사용자용 셀프 관리 앱`보다는 `체험단 운영 대행/도우미 서비스`에 더 가깝다.

확인된 특징:

- 전국 여러 플랫폼의 체험단을 한곳에 모아 검색
- 일정, 가이드, 메모 관리 언급
- 블로그 세팅, 글쓰기, 체험단 선정 보조 등 운영 대행 성격 강함

한계로 보이는 점:

- 공개 페이지 기준으로 사이트 계정 연동이나 선정 링크 자동 파싱 구조는 확인되지 않음
- 사용자가 스스로 선정 건을 구조화해 운영하는 SaaS라기보다 서비스형 운영 지원에 가까움

출처:

- 협찬플러스 공식 사이트

## 4. 해외에서 보인 유사 서비스

### Collabed

`Collabed`는 크리에이터용 브랜드 딜 관리 도구로, 보드, 간트, 캘린더, 데드라인, 할 일, 딜 상세 공간 등을 제공한다.

확인된 특징:

- 브랜드 딜 상태 관리
- 캘린더와 간트 차트
- 마감일과 할 일 관리
- 이메일/DM 기반 딜 생성
- 플랫폼별 협업 추적

우리 서비스와의 차이:

- 체험단 사이트 계정 연동 기반이 아님
- 국내 체험단 사이트의 선정 페이지 파싱 개념이 아님
- `브랜드 협업 CRM`에 가까움

출처:

- Collabed Features 페이지

### pageo

`pageo`는 크리에이터 CRM으로, 브랜드 문의, 딜 파이프라인, 결제 상태, 리마인더 등을 관리하는 서비스다.

확인된 특징:

- 브랜드 딜/문의 통합 inbox
- deal pipeline
- deliverables 및 payment tracking
- tasks with reminders

우리 서비스와의 차이:

- 체험단 사이트에서 당첨된 캠페인을 로그인 상태로 읽어오는 구조가 아님
- 딜 수집이 주로 문의/인바운드/연락 흐름 중심

출처:

- pageo 공식 사이트

### Influzivo

검색 결과 기준으로 `Influzivo`는 브랜드와 크리에이터가 캠페인을 관리하고, 보드/테이블/캘린더 관점으로 운영하는 플랫폼으로 보인다.

우리 서비스와의 차이:

- 체험단 선정 링크 자동 파싱보다 `브랜드-크리에이터 캠페인 운영 플랫폼`에 더 가까움
- 국내 체험단 사이트 구조 대응과는 결이 다름

출처:

- Influzivo 공식 사이트 검색 결과 설명

## 5. 시장에서 보이는 공백

검색 결과를 기준으로 보면 다음 공백이 보인다.

- 국내 체험단 사이트를 여러 개 연결해 `선정 이후 운영`을 한곳에서 관리하는 도구가 드물다
- `공고 탐색`은 있지만 `선정 링크 기반 자동 수집`은 잘 보이지 않는다
- `마감일 관리`는 있지만 `로그인 후 상세정보 자동 추출`은 잘 보이지 않는다
- 글로벌 creator CRM은 많지만 국내 체험단 워크플로우에 그대로 맞지 않는다

## 6. 우리 서비스의 차별화 포인트 후보

- 체험단 `공고 탐색`이 아니라 `선정 이후 관리`에 집중
- 사이트별 계정 연동 후 로그인 상태 파싱
- 선정 링크 등록만으로 상세 정보 자동 수집
- 체험 가능 기간과 리뷰 마감일을 캘린더로 통합
- 날짜 미정/확정 상태를 바로 관리
- 전화번호, 주소, 상세 안내 등 로그인 후 정보까지 관리

## 7. 현재 판단

현재 공개 검색 기준으로는 `완전히 동일한 경쟁 서비스`보다 `부분적으로 겹치는 서비스`가 더 많다.

그래서 이 프로젝트는 완전히 새로운 시장을 만드는 수준까지는 아니더라도, 적어도 아래 조합에서는 차별성이 있다.

- 국내 체험단 운영 흐름 특화
- 선정 이후 일정 관리 특화
- 로그인 기반 상세 정보 자동 수집 특화
